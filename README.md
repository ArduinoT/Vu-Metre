# Vu-Metre
Arduino ile Vu Metre

Merhaba arkadaşlar bugün sizlere bir Vu Metre yaptık. [ Buraya tıklayarak ]( https://www.youtube.com/channel/UCwGhWHqg4HuKZ0C9j4vjGxg/videos ) kanalıma göz atabilirsiniz.

## Malzemeler:
- Arduino UNO
- Breadboard
- Direnç
- Ses Sensörü
- 9 tane led
- Jumper Kablolar
